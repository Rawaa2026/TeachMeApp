شرح كود شاشة نشر درس جديد (PublishLessonActivity)

هذه الشاشة مخصصة للمعلمين فقط، حيث تسمح لهم بإدخال تفاصيل الدرس الجديد (المادة، العنوان، الوقت، والسعر) ونشره ليظهر للطلاب في التطبيق.

1. اختيار المادة (Subject Spinner)

عند بدء الشاشة، يتم ملء قائمة منسدلة (Spinner) بجميع المواد المتاحة للتدريس من ملف الموارد (Strings/Arrays). 

ArrayAdapter<CharSequence> adapter = ArrayAdapter.createFromResource(this,
        R.array.teacher_subjects, android.R.layout.simple_spinner_dropdown_item);
subjectSpinner.setAdapter(adapter);

2. اختيار التاريخ والوقت (DateTime Picker)

يوفر الكود تجربة مستخدم سلسة؛ عند الضغط على زر اختيار الوقت، يظهر أولاً تقويم لاختيار اليوم (DatePicker)، وفور الاختيار يظهر منبه لاختيار الساعة (TimePicker). ثم يتم تحويل الاختيار إلى تنسيق ZonedDateTime. 

private void showDateThenTimePicker() {
    DatePickerDialog dateDialog = new DatePickerDialog(this, (view, year, month, dayOfMonth) -> {
        // بعد اختيار التاريخ، يفتح اختيار الوقت مباشرة
        TimePickerDialog timeDialog = new TimePickerDialog(this, (tv, hourOfDay, minute) -> {
            // دمج التاريخ والوقت في كائن واحد
            LocalDateTime ldt = LocalDateTime.of(date, time);
            chosenStart = ldt.atZone(ZoneId.systemDefault());
            datetimeSummary.setText(formatChosen(chosenStart));
        }, ...);
        timeDialog.show();
    }, ...);
    dateDialog.show();
}

3. التحقق والنشر (submitLesson)

قبل إرسال البيانات لقاعدة البيانات، يقوم الكود بعدة فحوصات أمان:
  • التأكد من أن المستخدم مسجل كـ معلم.
  • التأكد من أن عنوان الدرس ليس فارغاً.
  • التأكد من اختيار تاريخ ووقت الدرس.
  • الأهم: التأكد من أن وقت الدرس في المستقبل وليس في الماضي.
إذا نجحت الفحوصات، يتم استدعاء LessonRepository لحفظ الدرس في Appwrite. 

private void submitLesson() {
    // التحقق من عنوان الدرس
    if (title.isEmpty()) { ... return; }

    // التحقق من أن الوقت في المستقبل
    if (!LessonRepository.isStartAtInFuture(iso)) {
        Toast.makeText(this, "يجب اختيار وقت في المستقبل", Toast.LENGTH_SHORT).show();
        return;
    }

    // إرسال البيانات للمستودع (Repository) للحفظ
    new LessonRepository(this).publishLesson(teacherId, subj, title, iso, price, result -> {
        if (result.success) {
            Toast.makeText(this, "تم نشر الدرس بنجاح", Toast.LENGTH_SHORT).show();
            finish(); // إغلاق الشاشة والعودة
        }
    });
}
