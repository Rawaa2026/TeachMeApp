2. الشاشة الرئيسية (MainActivity)

com.example.rawaaproject.MainActivity
تتحكم هذه الشاشة في واجهة المستخدم الأساسية. تقوم بفحص حالة تسجيل الدخول؛ إذا كان المستخدم مسجلاً، تعرض له القائمة السفلية (الرئيسية، المعلم، الطالب، الملف الشخصي) بناءً على دوره. 

public class MainActivity extends AppCompatActivity {
    protected void onCreate(Bundle savedInstanceState) {
        // فحص حالة الجلسة
        if (sessionManager.isLoggedIn()) {
            showMainContent();
        } else {
            showAuth();
        }
    }
}

3. مدير الجلسات (SessionManager)

com.example.rawaaproject.data.SessionManager
هذا الكلاس مسؤول عن تخزين بيانات المستخدم (مثل ID والبريد والدور) محلياً على الهاتف باستخدام SharedPreferences، لكي لا يحتاج المستخدم لتسجيل الدخول في كل مرة يفتح فيها التطبيق. 

public class SessionManager {
    public void saveUserSession(User user) {
        String userJson = gson.toJson(user);
        prefs.edit().putString(KEY_USER_JSON, userJson).apply();
    }
    public boolean isLoggedIn() {
        return prefs.getBoolean(KEY_LOGGED_IN, false);
    }
}
