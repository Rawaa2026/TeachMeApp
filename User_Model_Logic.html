شرح كود نموذج المستخدم (User Model)

كلاس User.java هو الهيكل الأساسي الذي يمثل أي مستخدم في تطبيق TeachMeApp. يحتوي هذا الملف على كافة الحقول اللازمة لتخزين بيانات المدرسين والطلاب على حد سواء، ويُستخدم لنقل البيانات بين واجهات التطبيق وقاعدة بيانات Appwrite.

1. الهوية والبيانات الأساسية

هذه الحقول تحدد هوية المستخدم ونوع حسابه داخل النظام.
  • id / userId: المعرفات الفريدة للمستخدم في قاعدة البيانات.
  • fullName: الاسم الكامل للمستخدم الذي سيظهر للآخرين.
  • email: البريد الإلكتروني (يُستخدم كمعرف للدخول).
  • role: يحدد نوع الحساب، ويأخذ قيمتين فقط: "teacher" أو "student".

public String id;
public String fullName;
public String email;
public String role; // "teacher" أو "student"

2. البيانات الخاصة بالأدوار

يتميز هذا النموذج بأنه "هجين"، حيث يحتوي على حقول خاصة بالمعلم وحقول أخرى خاصة بالطالب:
للمعلم (Teacher):
  • specialization: المواد التي يدرسها.
  • experience: سنوات الخبرة.
  • hourlyRate: سعر الساعة التعليمية.
للطالب (Student):
  • birthDate: تاريخ الميلاد (لحساب العمر).
  • grade: الصف الدراسي.
  • school: اسم المدرسة.

// حقول المعلم
public String specialization;
public String experience;
public String hourlyRate;

// حقول الطالب
public String birthDate;
public String grade;
public String school;

3. الدوال المساعدة (Helper Methods)

يحتوي الكود على دوال تسهل التعامل مع المستخدم برمجياً دون الحاجة لكتابة شروط معقدة في كل مكان: 

// التحقق من نوع المستخدم
public boolean isTeacher() { return "teacher".equals(role); }
public boolean isStudent() { return "student".equals(role); }

// جلب الاسم للعرض (إذا كان الاسم فارغاً يعرض البريد)
public String getDisplayName() {
    return fullName != null && !fullName.isEmpty() ? fullName : email;
}

// التحقق من صحة البيانات قبل إرسالها للسيرفر
public boolean isValid() {
    return email != null && fullName != null && role != null;
}

4. بيانات النظام والوقت

تُستخدم هذه الحقول لتتبع حالة الحساب وتاريخ إنشائه وتحديثه.
  • createdAt: وقت إنشاء الحساب (بصيغة Timestamp).
  • updatedAt: آخر وقت تم فيه تعديل بيانات الملف الشخصي.
  • isActive: حالة الحساب (مفعل أو معطل).
تم استخراج هذا التوثيق لنموذج المستخدم في مشروع TeachMeApp.
