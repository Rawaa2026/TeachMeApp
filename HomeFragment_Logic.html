شرح كود الشاشة الرئيسية (HomeFragment)

تعتبر هذه الشاشة هي "اللوحة الرئيسية" للمستخدم، حيث تعرض له جدول الحصص المعتمدة (سواء كان معلماً يريد رؤية حصصه التي سيقدمها، أو طالباً يريد رؤية حصصه التي سيتلقاها).

1. جلب بيانات الجدول (loadScheduleTable)

هذه الدالة هي المسؤولة عن تحديد ما سيظهر في الجدول بناءً على دور المستخدم الحالي:
  • إذا كان معلماً: يتم جلب الدروس التي وافق فيها على طلاب.
  • إذا كان طالباً: يتم جلب الدروس التي تم قبوله فيها.
العملية تتم عبر مستودع الدروس (LessonRepository). 

private void loadScheduleTable() {
    User u = sessionManager.getCurrentUser();
    if (u.isTeacher()) {
        // جلب جدول المعلم
        lessonRepo.loadTeacherSchedule(u.id, result -> {
            ArrayList<Lesson> list = result.data;
            bindScheduleCards(list, false, ...);
        });
    } else {
        // جلب جدول الطالب
        lessonRepo.loadStudentSchedule(u.id, result -> {
            // ... منطق إضافي لجلب أسماء المدرسين
            bindScheduleCards(list, true, ...);
        });
    }
}

2. جلب بيانات المعلمين في الخلفية (للطلاب)

بما أن كائن "الدرس" يحتوي فقط على معرف المعلم (teacherId)، يحتاج التطبيق عند دخول الطالب لجلب اسم ورقم هاتف المعلم من قاعدة البيانات. يتم ذلك في Thread منفصل لضمان سرعة التطبيق، ثم يتم تحديث الواجهة. 

new Thread(() -> {
    UserRepository ur = new UserRepository(requireContext());
    for (Lesson l : snapshot) {
        // جلب بيانات المعلم لكل درس بالمعرف الخاص به
        User t = ur.getUserById(l.teacherId);
        teacherNames.put(l.teacherId, t.getDisplayName());
        teacherPhones.put(l.teacherId, t.phoneNumber);
    }
    // العودة لواجهة المستخدم لتحديث البيانات
    root.post(() -> bindScheduleCards(...));
}).start();

3. ربط البيانات بالمحول (HomeScheduleAdapter)

هذه الدالة تقوم بإنشاء "المحول" (Adapter) الذي يأخذ قائمة الدروس ويحولها إلى "بطاقات" (Cards) مرئية في القائمة. كما تحتوي على "مستمع للضغطات" (Click Listener) للتعامل مع عمليات الإلغاء. 

private void bindScheduleCards(ArrayList<Lesson> lessons, boolean isStudent, ...) {
    HomeScheduleAdapter adapter = new HomeScheduleAdapter(
        lessons, isStudent, teacherNames, teacherPhones, ... ,
        (lesson, isStudentRole) -> {
            // منطق الإلغاء عند الضغط على الزر
            if (isStudentRole) {
                lessonRepo.cancelStudentEnrollment(lesson.id, ...);
            } else {
                lessonRepo.cancelLessonByTeacher(lesson.id, ...);
            }
        });
    scheduleRecycler.setAdapter(adapter);
}

4. منطق الإلغاء (Cancellation)

يوفر الكود ميزة إلغاء الحصة مباشرة من الصفحة الرئيسية:
  • للمعلم: إلغاء الدرس بالكامل.
  • للطالب: إلغاء تسجيله في هذا الدرس فقط.
بعد كل عملية إلغاء، يتم استدعاء loadScheduleTable() مجدداً لتحديث القائمة فوراً.
