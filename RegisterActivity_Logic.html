شرح كود شاشة التسجيل (RegisterActivity)

تعتبر شاشة RegisterActivity المسؤولة عن تسجيل مستخدمين جدد في تطبيق TeachMeApp، حيث تسمح للمستخدم باختيار دوره (معلم أو طالب) وإدخال بياناته.

1. اختيار الدور (setRole)

هذه الدالة مسؤولة عن التبديل بين وضع "المعلم" و "الطالب" بصرياً وبرمجياً:
  • تقوم بتغيير لون الإطار (Stroke) للبطاقة المختارة.
  • تقوم بتغيير لون زر "المتابعة" ليتناسب مع لون الدور (أزرق للطالب، أخضر للمعلم مثلاً).
  • تقوم بإظهار أو إخفاء قسم "تاريخ الميلاد" الخاص بالطلاب فقط.

private void setRole(boolean student) {
    isStudent = student;
    // تحديد لون التمييز بناءً على الدور
    int accent = student ? studentColor : teacherColor;

    // إظهار قسم تاريخ الميلاد للطلاب فقط
    studentBirthBlock.setVisibility(student ? View.VISIBLE : View.GONE);

    // تغيير شكل البطاقات (Cards) لتمييز الاختيار
    if (student) {
        cardStudent.setStrokeWidth(strokePx);
        cardTeacher.setStrokeWidth(0);
    } else {
        cardTeacher.setStrokeWidth(strokePx);
        cardStudent.setStrokeWidth(0);
    }
}

2. تهيئة اختيار تاريخ الميلاد (setupStudentBirthPickers)

تستخدم هذه الدالة كلاس مساعد BirthDateSpinners لملء القوائم المنسدلة (Spinners) بالسنوات والشهور والأيام، وتضمن تحديث عدد الأيام تلقائياً بناءً على الشهر والسنة المختارين. 

private void setupStudentBirthPickers() {
    BirthDateSpinners.setupYear(this, birthYearSpinner, ...);
    BirthDateSpinners.setupMonth(this, birthMonthSpinner, ...);
    BirthDateSpinners.setupDays(this, birthDaySpinner, ...);
    // ربط التحديثات التلقائية للأيام
    BirthDateSpinners.wireDayUpdates(this, birthYearSpinner, birthMonthSpinner, birthDaySpinner, ...);
}

3. منطق التحقق من البيانات (Validation)

قبل إرسال البيانات للسيرفر، يتم التأكد من:
  • أن جميع الحقول المطلوبة مملوءة (الاسم، الهاتف، البريد).
  • أن كلمة المرور لا تقل عن 6 أحرف وتطابق تأكيد كلمة المرور.
  • للطلاب: التأكد من اختيار تاريخ الميلاد وأن العمر لا يقل عن 7 سنوات.

if (password.length() < 6) {
    Toast.makeText(this, "كلمة المرور قصيرة جداً", Toast.LENGTH_SHORT).show();
    return;
}
if (isStudent) {
    String birthDate = BirthDateSpinners.toIsoDateOrNull(...);
    if (!ProfileValidation.isAgeAtLeastYears(birthDate, 7)) {
        Toast.makeText(this, "يجب أن يكون عمر الطالب 7 سنوات على الأقل", Toast.LENGTH_SHORT).show();
        return;
    }
}

4. تنفيذ عملية التسجيل (submitRegister)

عند الضغط على زر التسجيل، يتم استدعاء authRepository.register. في حال النجاح، يتم حفظ بيانات "الجلسة" محلياً وإغلاق الشاشة للانتقال للرئيسية. 

authRepository.register(role, name, email, password, phone, birthDate, ... result -> {
    if (result.success) {
        // حفظ الجلسة محلياً
        sessionManager.saveUserSession((User) result.customData);
        setResult(RESULT_REGISTER_OK);
        finish(); // إغلاق الشاشة
    } else {
        Toast.makeText(this, "فشل التسجيل: " + result.message, Toast.LENGTH_SHORT).show();
    }
});
تم استخراج هذا التوثيق البرمجي لشاشة التسجيل في مشروع TeachMeApp.
