توثيق حزمة واجهات المستخدم (UI Package) لمشروع TeachMeApp

انتقال سريع: الرئيسية المراكز (Hubs) الدروس الطلبات الملف الشخصي المحولات (Adapters)

1. الشاشة الرئيسية (HomeFragment)

HomeFragment.java هذه هي واجهة الهبوط الأولى للمستخدم بعد تسجيل الدخول. وظيفتها عرض جدول الحصص "المعتمدة" فقط لليوم وللمستقبل.
  • تقوم بفحص دور المستخدم (معلم أو طالب).
  • تجلب قائمة الدروس المقبولة من المستودع.
  • تعرض تنبيهاً في حال كان الجدول فارغاً.

private void loadScheduleTable() {
    User u = session.getCurrentUser();
    if (u.isTeacher()) {
        lessonRepo.loadTeacherSchedule(u.id, result -> {
            // عرض حصص المعلم المعتمدة
            bindScheduleCards(result.data, false, ...);
        });
    } else {
        lessonRepo.loadStudentSchedule(u.id, result -> {
            // عرض حصص الطالب المقبول فيها
            bindScheduleCards(result.data, true, ...);
        });
    }
}

2. مراكز التحكم (StudentHub & TeacherHub)

StudentHubFragment.java & TeacherHubFragment.java هذه الملفات تستخدم نظام ViewPager2 و TabLayout لتنظيم المحتوى.
  • مركز الطالب: يحتوي على تبويبات (تصفح الدروس، البحث عن مدرسين، الجدول الزمني).
  • مركز المعلم: يحتوي على تبويبات (دروسي، الجدول الزمني، طلبات الانضمام).

// كود مشترك لتنظيم التبويبات (Tabs)
pager.setAdapter(new FragmentStateAdapter(this) {
    public Fragment createFragment(int position) {
        if (position == 0) return new StudentLessonsFragment();
        if (position == 1) return new StudentTeachersFragment();
        return new ScheduleFragment();
    }
    public int getItemCount() { return 3; }
});

3. إدارة تصفح الدروس (StudentLessonsFragment)

StudentLessonsFragment.java هذه الشاشة هي "سوق الدروس" للطالب.
  • تسمح بتصفية الدروس حسب المادة (Spinner).
  • تعرض حالة كل درس (طلب انضمام، قيد المراجعة، تم القبول).
  • تسمح للطالب بإرسال طلب انضمام جديد بنقرة زر.

private void requestJoin(Lesson lesson) {
    repo.requestJoin(lesson.id, u.id, r -> {
        Toast.makeText(getContext(), "تم إرسال الطلب", Toast.LENGTH_SHORT).show();
        reload(); // تحديث القائمة فوراً
    });
}

4. إدارة طلبات الطلاب (TeacherRequestsFragment)

TeacherRequestsFragment.java واجهة خاصة بالمعلم لمراجعة الطلاب الذين يريدون الانضمام لدروسه.
  • يعرض اسم الطالب والدرس المطلوب.
  • يوفر زرين: قبول (Approve) أو رفض (Reject).
  • عند القبول، تظهر الحصة تلقائياً في جدول الطالب والمعلم في الصفحة الرئيسية.

private void approve(LessonEnrollment e) {
    repo.setEnrollmentStatus(e.id, LessonEnrollment.STATUS_APPROVED, u.id, r -> {
        Toast.makeText(getContext(), "تم قبول الطالب", Toast.LENGTH_SHORT).show();
        load(); // تحديث القائمة
    });
}

5. الملف الشخصي (ProfileFragment)

ProfileFragment.java عرض بيانات المستخدم الشاملة.
  • يعرض الصورة الشخصية والبيانات الأساسية.
  • يعرض إحصائيات (الأرباح للمعلم / التكاليف للطالب).
  • يوفر خيار تسجيل الخروج وتعديل البيانات.

6. محولات البيانات (Adapters)

HomeScheduleAdapter.java هذا الكلاس هو المسؤول عن "رسم" كل بطاقة درس في القوائم.
  • يحدد لون البطاقة وحالة الأزرار.
  • يضع علامة "اليوم" على الحصص الحالية.
  • يشطب على الحصص القديمة (Strike-through).

public void onBindViewHolder(@NonNull VH h, int position) {
    Lesson l = lessons.get(position);
    // تنسيق شكل البطاقة بناءً على حالة الدرس ووقت بدايته
    LessonScheduleCardStyle.apply(h.card, l);
    h.time.setText(LessonUiHelper.formatStartAt(l.startAt));

    // إخفاء أو إظهار زر الإلغاء بناءً على الوقت المتبقي
    boolean canCancel = LessonRepository.canCancelBeforeLessonStart(l.startAt);
    h.cancel.setVisibility(canCancel ? View.VISIBLE : View.GONE);
}
